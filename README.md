# 音乐下载器

2026年3月最新的免VIP音乐破解下载工具，支持通过歌名和歌手名搜索并下载 MP3 音乐文件。

## 功能特性

### 命令行版本
- 根据歌名或歌手名搜索音乐
- 显示搜索结果列表（歌名对齐显示）
- 选择并下载指定音乐
- 自动以"歌名 - 歌手.mp3"格式保存文件

### GUI版本
- 提供图形用户界面，操作更直观
- 支持批量输入歌曲信息（每行格式：歌名-歌手 或 歌名 - 歌手）
- 自动搜索并匹配歌曲
- 对于不匹配的歌曲，提供搜索结果选择功能
- 支持批量下载歌曲到music文件夹
- 下载失败时自动重试（最多3次）
- 显示下载进度和结果统计

## 依赖要求

- Python 3.x
- requests
- lxml
- tkinter（GUI版本，Python标准库）

## 安装

1. 克隆或下载本项目
2. 安装依赖：

```bash
pip install requests lxml
```

## 使用方法

### 命令行版本

运行主程序：

```bash
python main.py
```

#### 操作步骤

1. 输入要搜索的歌名或歌手名
2. 查看搜索结果列表
3. 输入要下载的歌曲编号
4. 等待下载完成

#### 示例

```
请输入歌名或歌手: 周杰伦
1. 晴                  周杰伦
2. 稻香                周杰伦
3. 青花瓷              周杰伦
请输入编号下载: 1
开始下载: 晴 - 周杰伦.mp3
已下载: 晴 - 周杰伦.mp3
```

### GUI版本

运行主程序：

```bash
python tk_gui.py
```

#### 操作步骤

1. 在输入框中输入歌曲信息，每行格式为"歌名-歌手"或"歌名 - 歌手"
2. 点击"提取歌曲"按钮开始搜索和匹配
3. 对于不匹配的歌曲，在弹出的对话框中选择要添加的歌曲
4. 提取完成后，点击"下载歌曲"按钮将所有歌曲下载到music文件夹
5. 下载完成后会显示成功和失败的统计信息
